# minesweeper

Para correr: node app.js

Reglas:

1. Seleccionar la cantidad de bombas y el tamaño del tablero.
2. Calcular con el botón derecho donde podrían estar las bombas.

Se puede reiniciar el juego con los valores seteados al principio o se puede modificar el tamaño del tablero.

## También se puede guardar la partida
Por el momento la partida se guarda en txt en la ubicación que el/la/le jugador/a/e elija. Para recuperar seleccionar archivo.
